A fonte utilizada para a criação é a Shine, desenvolvida pelo designer @billyargel.

Segoe UI


Desenvolvida pela Microsoft, a Segoe UI é a fonte principal do nosso sistema. Uma fonte sem serifa que prioriza a legibilidade, proporcionando uma melhor experiência para nossos usuários.


Segoe UI bold


A versão Bold é utilizada em títulos e palavras em destaque durante toda navegação


Segoe UI Regular


A versão regular é utilizadas em caixas de textos e outras informações dos usuários
